# Tower Defense Game

A simple JavaScript tower defense game with 3 enemy types and 2 tower types. More content coming soon.

## How to Play

Select towers from the 6-slot panel on the right and press play.

## Requirements

- PHP 8.4.6

## Local Development

```bash
tsc
php -S localhost:9000
```
